# 🎯 Bit-54 Activities

**48 hands-on activities for BBC micro:bit** — from blinking LEDs to BLE robots and AI, beginner to advanced.

## Quick Start

Open `index.html` in any browser — no install, no server.

## What's Inside

- 48 guided activities across multiple difficulty levels
- Covers LEDs, buttons, sensors, radio, BLE, robotics, AI and more
- Trilingual interface (EN/FR/AR)
- Single-file HTML app, zero dependencies
